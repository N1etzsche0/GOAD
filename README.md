# 域渗透靶场GOAD 学习笔记

## 域渗透的重要性

因为OSCP考试的改革，域渗透部分占到40分，所以域渗透重要性不言而喻

而针对域渗透，最好的方法就是多加练习!

GOAD域渗透靶场，应该是目前最好的靶场，我曾多次在HTB靶机上看到相似部分

计划在暑假结束前打通GOAD靶场，对域渗透有个清晰的认识
